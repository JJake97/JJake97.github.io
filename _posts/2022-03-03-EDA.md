# Exploratory Data Analysis (EDA)

- 데이터셋을 다양한 관전에서 살펴보고 탐색하면서 인사이트를 찾는 것
- 각 행과 열이 무엇을 의미하는가?
- 각 column은 어떤 분포를 보이는가?
- 두 column은 어떤 연관성이 있는가?

- 시각적인 방법, 통계적인 방법 등 데이터를 파악할 수 있는 모든 방법이 동원된다


```python
%matplotlib inline

import seaborn as sns
```

